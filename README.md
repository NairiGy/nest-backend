## Installation

```bash

$  npm  install

```

## Running the app using docker

```bash

$ docker-compose up

// Postgres -> localhost:5432
// nest server -> localhost:3000

```

## Test

```bash

# unit tests

$  npm  run  test


# test coverage

$  npm  run  test:cov

```
